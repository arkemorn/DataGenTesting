Данный код создан для ознакомительной практики.

Для выбора сферы (их 3) введите число от 1 до 3. Если выберете другое - программа выдаст ошибку и нужно будет запускать заново.
Для выбора при желании примера 5-ти сгенерированных данных введите число от 1-6. При выборе другого программа попросит ввести заново. При вводе нуля программа остановится.
