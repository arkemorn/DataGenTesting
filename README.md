Данный код создан для ознакомительной практики.

Для выбора сферы (их 3) введите число от 1 до 4, 4 - вторая версия данных. Если выберете другое - программа выдаст ошибку и нужно будет запускать заново.
Для выбора при желании примера 5-ти сгенерированных данных введите число от 1-6, 7 - для проверки поиска в SQL. При выборе другого программа попросит ввести заново. При вводе нуля программа остановится и выведет на экран графики сравнений (Может отобразиться неправильно, лечится двумя нажатиемя на смену режима отображения экрана), а так же сохранит картинку с графиками и откроет папку в проводнике с созданными файлами.
